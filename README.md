# AI Agent in C# (Unit Test Benchmark)

ยินดีต้อนรับสู่ Repository หลักของโปรเจกต์ **AI Agent Unit Test Benchmark** ครับ! โปรเจกต์นี้ถูกสร้างขึ้นมาเพื่อใช้ทดสอบและประเมินประสิทธิภาพของ AI ในการเขียน Unit Test สำหรับภาษา C# โดยเฉพาะ

## 📂 โครงสร้างของโปรเจกต์ (Project Structure)

โปรเจกต์นี้ถูกแบ่งออกเป็น 2 ส่วนหลักๆ ดังนี้ครับ:

1. **[`ai-unit-test-benchmark/`](./ai-unit-test-benchmark/)** 🐍
   - เป็นสคริปต์ Python หลักที่ใช้ทำ Benchmark
   - มีชุดข้อมูล (Dataset) และระบบประเมินผล AI Agent

2. **[`ai-unit-test-app/`](./ai-unit-test-app/)** 🌐
   - เป็น Web Application สำหรับใช้งานระบบประเมินผล
   - ประกอบไปด้วยส่วน Frontend และ Backend

## 🔗 โปรเจกต์ที่ใช้สำหรับทดสอบ (Demo Projects)
ตัวอย่างโปรเจกต์ภาษา C# ที่เราใช้เป็นโจทย์ในการทดสอบ AI ได้ถูกแยก Repository ออกไปเพื่อให้ระบบ Clean ที่สุดครับ:
- [Employee Management Demo](https://github.com/stampfordsit/ai-demo-employee-management-demo)
- [Calculator Service Demo](https://github.com/stampfordsit/ai-demo-calculator-service-demo)
